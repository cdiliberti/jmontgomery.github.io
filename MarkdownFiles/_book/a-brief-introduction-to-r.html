4 A Brief Introduction to R

Learning objectives:

  1. Learn how to assign a value to an object in R.

  2. Learn your way around R (using R Studio).

  3. Learn how to do simple arithmetic in R.

4.1 First follow along with this video

4.2 Now see if you can do this on your own

Here are some basic things to try in R:

  • Multiplying, dividing, adding numbers in Rstudio console:
5*3
5+3
5/3
  • Assigning a value to an object: (This assigns x the value of 6)
x <- 6
  • Install a package:
install.packages("foreign")
  • Call a package through its library:
library(foreign)

This CRAN-package, foreign, will come handy when you read data into R next week!
